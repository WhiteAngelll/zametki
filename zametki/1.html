Удалить заметку из корзины

Метод basket_delete (DELETE) Окончательно удаляет заметку без возможности восстановления. Что бы узнать id нужной заметки, воспользуйтесь методом "basket_list". http://194.58.108.107/api/basket/{{note_basket_id}}/
Параметры
  • {{access}} - access токен
  • {{access}} - access токен

Request example

Header

cURL

curl -X DELETE 'http://194.58.108.107/api/basket/19/' \ -H 'Authorization: Bearer {{access}}'
